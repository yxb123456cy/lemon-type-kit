# 简介

Lemon Type Kit 是一个现代化的 TypeScript 工具库，旨在提供全面、高效且类型安全的实用函数，帮助开发者更轻松地构建应用程序。

## 为什么选择 Lemon Type Kit？

- **类型安全**: 完全使用 TypeScript 编写，提供优秀的类型推断和提示。
- **模块化**: 函数按功能模块组织（如 Array, String, Number 等），方便按需引用。
- **零依赖**: 核心库不依赖任何第三方运行时库，保持轻量级。
- **高性能**: 经过优化的实现，确保在各种场景下的高效运行。
- **文档齐全**: 提供详细的 API 文档和使用示例。
- **Tree Shaking**: 支持 Tree Shaking，确保打包体积最小化。

## 包含模块

目前包含以下核心模块：

- **Array Utils**: 数组操作工具（去重、分块、打乱等）
- **String Utils**: 字符串处理工具（大小写转换、截断、模板替换等）
- **Number Utils**: 数值计算工具（范围限制、随机数、统计等）
- **Date Utils**: 日期时间处理工具
- **Cache Utils**: 简单的内存缓存工具
- **Log Utils**: 控制台日志增强工具
- **Phone Utils**: 手机号处理工具
